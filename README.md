# Guru99 Bank Manual Testing

This repository contains the testing artifacts created for the Guru99 Bank web application.

## Project Overview

Guru99 Bank is a web-based banking application that provides banking services for Managers and Customers. This project focuses on manual testing of the application's core functionalities.

## Modules Tested

* Login
* New Customer
* Edit Customer
* New Account
* Edit Account
* Delete Account
* Delete Customer
* Balance Enquiry
* Mini Statement
* Customized Statement

## Testing Types

* Functional Testing
* Retesting
* Regression Testing

## Tools Used

* Google Sheets
* Chrome Browser
* Brave Browser

## Objective

The objective of this project is to practice requirement analysis, test case design, test execution, defect reporting, retesting, and regression testing for a banking application.
